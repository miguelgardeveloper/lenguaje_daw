¿Qué es Markdown?

Markdown es un lenguaje de marcas ligero que se utiliza para crear texo formateadoutilizando un editor de texto plano. Se caracteriza por su simplicidad y su capacidad para ser convertido fácilmente a otros formatos, como HTML o PDF.
